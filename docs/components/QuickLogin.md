# QuickLogin 快速登录

## 介绍

QuickLogin 是一个快速登录组件，提供便捷的登录入口。

## 使用示例

### 基础用法

```vue
<script setup lang="ts">
import { QuickLogin } from '@pro-element-plus/components'
</script>

<template>
  <QuickLogin />
</template>
```
