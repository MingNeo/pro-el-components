# SideBar 侧边栏

## 介绍

SideBar 是一个侧边栏组件，用于应用的导航菜单展示。

## 使用示例

### 基础用法

```vue
<template>
  <ProSideBar>
    <template #default>
      <!-- 侧边栏内容 -->
    </template>
  </ProSideBar>
</template>
```

## API

### Props

| 属性名 | 说明 | 类型 | 默认值 |
| ------ | ---- | ---- | ------ |
| -      | -    | -    | -      |

### Slots

| 插槽名  | 说明                   |
| ------- | ---------------------- |
| default | 默认插槽，用于放置内容 |
